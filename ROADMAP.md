For this project, instead of starting to work with actual AI APIs to create my German AI Assistant, I will start by creating a non-AI prototype. This will ensure I have a good base of what I want this application to do before upgrading it to actual AI in the future.



v.0.0.1: For this version I want to simply load data from custom .csv files that I will add more to over time (for verbs, adjectives, etc.).

v.0.1.0: For this version I want to be able to question the user on different verbs, nouns and adjectives.
